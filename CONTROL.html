Project 1 : ROBOTY

This project is small demo for realizing a small mobile robot with 2 DC motors 6v controled by L298 circuit & arduino uno & bluetooth module HC06, ROBOTY app.

the components used are:
* arduino uno.
* HC06.
* 2 DC motors 6V & their power supply.
* L298.
* Voltage Regulator KA 7805.
* capacitor 22uF & 0.1uF
* 2 resistors 47kohm & 2 resistors 140ohm.
* 8 diode.
* 4 LED.
* batery 9V for arduino power supply.

connect every thing like the diagram below:

For full resolution download the diagram from here: DIAGRAM



The android app for this project can controle the robot's movement & speed.

the user can change the wheel & the background of the app (6 models).

When you run the app or the first time & for paired the bluetooth, the device aske you to enter PIN code, the default code is:1234

The system is protected by password saved in the arduino program so you can change it as you like without changing anything in the app. (in this program i make the password:9999)
You should activate the system by entering the true password after connecting the bluetooth to access the robot control.
Ensure that the bluetooth pin are disconnect with the arduino when you upload the arduino code;
You can control the robot by 3 way:
Click in the wheel; Drag the wheel; Rotate the phone.
* Roboty application download it from here: Roboty.apk
* The arduino code, download it from here: Roboty.ino

Software:
* XAMPP where to put the server program
* PYTHON 2.7:receive data from server & send order to arduino
* Pyserial

Hardware:
* arduino uno
* RC car & 4 diods

the server devide in 2 part:
* INDEXcar.php
* DATAROBOT.php

the server page must be a PHP page (exp:INDEXcar.php):
INDEXcar.php
DATAROBOT.txt
DATAROBOT.php
