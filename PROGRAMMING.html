For computer vision & image processing you need OpenCV to make your program easy & small:
* OpenCV-2.4.1:Download OpenCV for windows from: here
for opencv 2.4.1 or Higher i recommend to use it you need to do small trick for configuration:

*1 extract the file & save it in C(disk)
*2 Copy “cv2.pyd “ from C:\opencv\build\python\x86\2.7\
*3 paste it in C:\Python27\Lib\site-packages\

after that open python widow and type:
import cv2
if no error happen so all thing is good.
for opencv 2.2.0
* Add “C:/Python2.7;C:/OpenCV2.2/bin” to PATH variable .
* Create PYTHONPATH variable & put “C:/OpenCV2.2/Python2.7/Lib/site-packages” as value.
after that open python widow and type:
import cv
if no error happen so all thing is good.
